Tools for building coordinates of hetero compounds

The topology and parameter files for the acetyl and amide capping groups shown in the previous tutorial were created without the knowledge of atomic coordinates. This is only possible for very small compounds. For the vast majority of hetero compounds, it is necessary to know the atomic coordinates from which the topology and parameters can then be derived.

Obtaining existing coordinates

In favorable cases, atomic coordinates for a given hetero compound, or at least for a similar compound, can be found in the PDB Ligand Expo. The same compounds are also available through other web sites, which are often easier to use. Existing heterogens should be used whenever possible to keep consistent nomenclature.

Gerard Kleywegt's HIC-UP server maintains reference coordinates and CNS topology and parameter files for all heterogens in the PDB. It also provides numerous links and related information about each compound. The parameters are generated using XPLO2D, and are NOT validated.

EBI PDBeChem site (formerly MSDChem) maintains both PDB and MOL format coordinate files for all existing PDB heterogens. It has search tools to find matching or similar compounds, which seems to work better than the Ligand Expo search tool. (Note: HIC-UP includes links to matching PDBeChem entries, but PDBeChem is updated more frequently.)

Cambridge Structural Database (CSD) is the most comprehensive resource for small molecules structures. At the time of this writing, the CSD contains crystal structure information for over 460,000 organic and metal organic compounds. The coordinates are in general of high quality and very well suited for the derivation of topology and parameter definitions. However, the CSD is a commercial product and therefore its use is limited to members of registered, paying institutions.

Modelling coordinates from scratch

Some of the available parameterization tools support energy minimization using advanced force-fields or quantum-mechanical optimization. In most cases, the optimized coordinates are accurate enough to generate reasonable parameters. However, the are also prone to errors.

The PDB Ligand Expo search page includes MarvinSketch (press the [Launch] button). After sketching the molecule in 2D, select View -> Open in MarvinView3D. It will generate 3D coordinates and display them in a new window, from which you can save the coordinates in various formats, including PDB and MOL format. The resulting 3D coordinates are only qualitatively accurate, and must be optimized before generating parameters.

Corina generates 3D coordinates from a SMILES structure description, performs energy minimization, and returns the coordinates in PDB format. You can create the SMILES string using the JME Molecular Editor, available from many sites, or directly from Molinspiration. Just sketch a molecule, then click the smiley-face button.
