Cis peptides

Prolines

By default, the standard CNS protein topology and parameter files (protein.top, protein_rep.param, protein.param) recognize both cis and trans proline residues. The reason is that the omega dihedral angle is set to a two-fold potential (e.g., see the entry for "dihe CH1E C N CH1E" in CNS library protein_rep.param). In contrast, the CNS all hydrogen protein topology and parameter files fix the XXX-PRO peptide bond in the trans conformation. The reason is that the all hydrogen files are also used for NMR structure determination where it is necessary to lock the peptide geometry into either trans or cis.

Cis peptide bonds for specific residues

To set cis conformations for residues (other than proline for the standard parameter and topology files or for any residue for the all hydrogen files), a special parameter file needs to be generated using the CNS task file cis_peptide.inp. These parameters will be specific to the specified residues and lock the specified residue in the cis conformation.
