Distributed topology and parameter files

This section describes the parameter and topology files that are distributed with CNS. Topology and parameter files together represents a particular force field or empirical energy function. Files are supplied for proteins, nucleic acids, carbohydrates, water and ions. Some topology and parameter files contain polar hydrogen atoms (i.e.,exchangeable hydrogen atoms that can form hydrogen bonds) explicitly, whereas aliphatic hydrogen atoms are either explicitly included or implicitly modeled by appropriate modification of the heavy-atom van der Waals parameters. There are also files that contain all hydrogen atoms.

For most crystallographic applications hydrogen atoms are typically not used. Although most topology and parameter files contain at least polar hydrogen atoms, they are generally ignored since the coordinates are unknown. However, it is possible to add hydrogen atoms using the generate.inp or generate_easy.inp input files in the "general" section. Then, crystallographic refinement can be performed in the presence of polar (or all hydrogen atoms if the corresponding all hydrogen topology and parameter files are used).

 

Standard protein and nucleic acid files (with polar hydrogen atoms)

Protein

Note, that these standard topology and parameter files allow both cis and trans XXX-PRO peptide bonds. If other cis peptide bonds are present, the cns_peptide.inp task file must be used in order to generate a special parameter file (usually called "cis_peptide.param") that must be read as the last file in the list of parameter files.

protein.top

This describes the topology for protein building blocks (amino acids) including polar hydrogen atoms It also contains patches for peptide linkages, N- and C-terminal ending groups, the disulfide bond and histidine protonation states.

protein.link

This describes the linkage between protein building blocks (amino acids). It also contains the information about which N- and C-terminal ending groups are to be applied. This is not specific to crystallography or NMR.

protein_rep.param

This parameter file describes the parameters for proteins and should be used in conjunction with other parameter files which use a repulsive force field (protein_rep.param and dna-rna_rep.param). This is the suggested parameter set for crystallographic refinement if no hydrogen atoms are included in the model.

protein.param

This parameter file describes the parameters for proteins and should be used in conjunction with other parameter files which do not use a repulsive force field (protein.param and dna-rna.param).

protein_break.top

This describes patches for the deletion of peptide linkages.

Nucleic acids

dna-rna.top

This describes the topology for nucleic acid building blocks (nucleotides) including polar hydrogen atoms It also contains patches for nucleotide linkages, and 3' and 5'-terminal ending groups.

dna-rna.link

This describes the linkage between nucleic acid building blocks (nucleotides). It also contains the information about which 3' and 5'-terminal ending groups are to be applied. Neither 3' or 5' ends of the nucleic acid have a phosphate attached.

dna-rna-pho.link

This describes the linkage between nucleic acid building blocks (nucleotides). It also contains the information about which 3' and 5'-terminal ending groups are to be applied. A phosphate group is applied to the 5' end of the nucleic acid.

dna-rna_rep.param

This parameter file describes the parameters for nucleic acids and should be used in conjunction with other parameter files which use a repulsive force field (protein_rep.param and dna-rna_rep.param). This is the suggested parameter set for crystallographic refinement if no hydrogen atoms are included in the model.

dna-rna.param

This parameter file describes the parameters for nucleic acids and should be used in conjunction with other parameter files which do not use a repulsive force field (protein.param and dna-rna.param).

 

Protein and nucleic acid files with all hydrogen atoms

Protein

Note, that the CNS all hydrogen topology and parameter files fix the XXX-PRO peptide bond in the trans conformation. If cis prolines or other cis peptide bonds are present, the cns_peptide.inp task file must be used in order to generate a special parameter file (usually called "cis_peptide.param") that must be read as the last file in the list of parameter files. It is generally advisable to run the task file "model_stats.inp" on the starting model and make sure that there are no large improper violations or angle violations.

protein-allhdg5-4.top

These files describe the topology for protein building blocks (amino acids) including all hydrogen atoms It also contains patches for peptide linkages, N- and C-terminal ending groups, the disulfide bond, histidine protonation states and L- to D-amino acid conversion.

protein-allhdg5-4.param

These parameter files describe the parameters for proteins and should be used in conjunction with other parameter files which use an all atom force field.

protein-allhdg5-4.link

protein-allhdg5-4-noter.link

protein-allhdg5-4-noCter.link

protein-allhdg5-4-noNter.link

These files describe the linkage between protein building blocks (amino acids). It also contains the information about which N- and C-terminal ending groups are to be applied.

Nucleic acids

dna-rna-allatom-hj-opls.top

These files describe the topology for nucleic acid building blocks (nucleotides) including all hydrogen atoms It also contains patches for nucleotide linkages, and 3' and 5'-terminal ending groups.

dna-rna-allatom-hj-opls.param

These parameter files describe the parameters for nucleic acids and should be used in conjunction with other parameter files which use an all atom force field.

dna-rna.link

This describes the linkage between nucleic acid building blocks (nucleotides). It also contains the information about which 3' and 5'-terminal ending groups are to be applied. Neither 3' or 5' ends of the nucleic acid have a phosphate attached.

dna-rna-pho.link

This describes the linkage between nucleic acid building blocks (nucleotides). It also contains the information about which 3' and 5'-terminal ending groups are to be applied. A phosphate group is applied to the 5' end of the nucleic acid.

 

Carbohydrate

carbohydrate.top

This describes the topology for carbohydrate building blocks (sugars) including all hydrogen atoms. It also contains patches for linkages between different sugars.

carbohydrate.param

This parameter file describes the parameters for carbohydrates and can be used in conjunction with any other parameter files. The parameters may need modification to be consistent with other parameter sets (in particular the force constants for impropers and dihedrals are relatively low compared to the parameter sets that use a repulsive nonbonded potential function).

 

Ions

ion.top

This describes the topology for many elements and ionic species of elements. Topology definitions also exist for ions such as sulphate, phosphate and tungstate.

ion.param

This parameter file describes the parameters for ions and can be used in conjunction with any other parameter files.

 

Water

water.top

This describes the water molecule topology to be used in combination with polar hydrogen protein and nucleic acid files.

water_rep.param

This parameter file can be used in conjunction with other parameter files which use a repulsive force field (protein_rep.param and dna-rna_rep.param). This is the suggested file for crystallographic refinement if no hydrogen atoms are included.

water.param

This describes the water molecule parameter file that is to be used in combination with polar hydrogen protein and nucleic acid files.

water-allhdg5-4.top

This describes the water molecule topology file that is to be used in combination with all hydrogen protein and nucleic acid files.

water-allhdg5-4.param

This describes the water molecule parameter file that is to be used in combination with all hydrogen protein and nucleic acid files.

 

Fluorescent dyes

dyes.top

dyes.param

These files contain topology and parameter information for the Cy3 and Cy5 fluorescent dyes, along with patch residues to link them to CYS residues.

 

Pseudo atoms

pseudo.top

pseudo.param

These files contain topology and parameter information for pseudo atoms that are used in docking calculations with FRET data.

 

Single atoms

single_atoms.top

single_atoms.param

These files contain topology and parameter information for isolated carbon and oxygen atoms to be used to provide a scattering model when the topology is unknown: example molecular replacement with a cryo EM mas

 

Axis system

axis.top

axis.param

These files contain topology and parameter information for the axis system required for NMR residual dipolar couplings.

 

Force field settings

The default nonbonded parameters and enegy term flags can be modified by reading a force field settings file in many CNS input files.

electrostatics.settings

This force field setings file should be used in conjunction with the protein-allhdg5-4.* and dna-rna-allatom-hj.* parameter and topology files. It turns on the electrostatics terms. Reference: J.P. Linge, M.A. Williams, C.A.E.M. Spronk, A.M.J.J. Bonvin, M. Nilges, Refinement of protein structures in explicit solvent. Proteins 50, 496-506, 2003.

 

Other information

Van der Waals radii

The sigma values in the parameter file protein-allhdg.param are different from those used in the other all-hydrogen force fields. The changes were made to make the van der Waals radii more consistent with those used by other NMR refinement programs. The sigma values have been set such that the van der Waals radii will be exactly like those used by the DISGEO program (Havel and Wuthrich 1984) when REPEl is set to 0.8. In addition, van der Waals radii of potential hydrogen bond donor-acceptor pairs have been reduced by appropriate NBFIx statements. The actual van der Waals radii used for different values of REPEl are given in the table below (in A).

H 1.250 1.125 1.0 0.938
C 1.875 1.688 1.5 1.406
N 1.688 1.519 1.350 1.266
