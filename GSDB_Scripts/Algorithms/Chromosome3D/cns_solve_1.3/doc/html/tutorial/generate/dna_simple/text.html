A simple DNA duplex

Here the input model is a DNA structure obtained from the PDB. A file describing the molecular topology of the model (MTF file) will be generated with the CNS task file generate_easy.inp:

      cns_solve < generate_easy.inp > generate_easy.out

Note that the selection to build atoms is set to "(none)". The task file can build unknown atoms by specifying "(not known)". This feature should be used with caution and the resulting structure carefully checked.
