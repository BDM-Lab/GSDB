A protein monomer + DNA fragment + ytterbium ion + water molecules

The structure with the PDB ID code 2bop is a protein monomer with a DNA fragment, a ytterbium ion, and water molecules.

The command to generate the files generate_easy.mtf and generate_easy.pdb is: 

      cns_solve < generate_easy.inp > generate_easy.out
