Intensity statistics

To check the completeness and the quality of the data, various intensity statistics can be computed with the analyse.inp task file. 
      cns_solve < analyse.inp > analyse.out
The results can be found in individual output files for each data set and each pair of data sets.
analyse_intensity_f_w?.distr
analyse_intensity_f_w?.histo

Diffraction ratio distributions and histograms of intensities. Various statistics are computed separately for acentric and centric reflections.
analyse_f_w?.wilson

Wilson plots, scaling factors and overall B-factors.
analyse_anom_diff_f_w?.distr
analyse_anom_diff_f_w?.histo

Diffraction ratio distributions and histograms of anomalous differences between F+ and F-.
analyse_diff_f_w?_f_w?.distr
analyse_diff_f_w?_f_w?.histo

Diffraction ratio distributions and histograms of dispersive (or isomorphous) differences.
analyse.matrix

Overall anomalous and dispersive or isomorphous differences for all data sets organized in one matrix.
