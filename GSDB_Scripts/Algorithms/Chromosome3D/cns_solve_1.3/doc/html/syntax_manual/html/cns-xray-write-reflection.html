write reflection {<xray-write-reflection-statement>} end


<xray-write-reflection-statement> :==

    output=<filename>
       Output file for reflections.
    selection=<xray-selection>
       Select reflections to be written. Default=(all)
    {<complex-object>|<real-object>|<integer-object>}
       If no objects are specifed all currently active objects are written.
