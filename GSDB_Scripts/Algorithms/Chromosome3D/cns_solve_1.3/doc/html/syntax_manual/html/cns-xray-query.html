query <xray-query-statement> end


<xray-query-statement> :==

    domain=reciprocal|real
       Domain to query. Default=none
    name=<xray-object>
       Name to query. Default=none

       The query will create the following symbols:
         $object_exist (<logical>) - true if object exists.
         $object_type (<xray-type>) - real, complex, or integer.
         $object_used (<logical>) - true if object is # 0.
         $object_group (<integer>) - reciprocal space object group id.
         $object_gtype (<xray-group-type>) - reciprocal space object group type.
         $object_gord (<integer>) - reciprocal space object group order.
       The last two are only created for existing reciprocal space objects.
