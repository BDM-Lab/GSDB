connectivity {<connectivity-statement>} end


   Determines the covalent connectivity of a specified set of atoms.


<connectivity-statement> :==

    selection=<atom-selection>
       Atom selection for connectivity calculation.
    print=<logical>
       If true prints information about sets.
    setidto=<atom-object>
       Each set is assigned a unique id (1,...,mset). The individual
       set ids are optionally stored in the specified atom-object.
    nsetto=<atom-object>
       The size of each set is stored in the specified atom-object.

       The total number of sets is stored in symbol $mset.
       All atomic properties are allowed except scatter_fp and scatter_fdp.
