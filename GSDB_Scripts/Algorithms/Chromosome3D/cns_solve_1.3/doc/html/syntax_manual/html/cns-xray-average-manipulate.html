<xray-average-manipulate-statement> :==

    mask=<real-space-object>
       Real space object containing the mask to be transformed. The
       transformed mask overwrites this object.
    [map=<real-space-object>]
       Real space object that will be transformed on the region
       delineated by the mask. The transformed real space object overwrites\
       this object.
    <matrix>
       Define the rotation matrix to be applied.
    [center=<vector>]
       Define the center of rotation. Default=(0,0,0)
    translation=<vector>
       Define the translation vector.
