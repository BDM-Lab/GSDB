lidens {<xray-lidens-statement>} end


<xray-lidens-statement> :==

    density=<real-space-object>
       no default
    atomselection=<atom-selection>
       Selected atoms for density interpolation. Default=(none)
    to=<atom-object>
       Define destination for interpolated density
    fractional=<logical>
       Input site is in fractional coordinates. Default=false
    site=<real> <real> <real>
       Coordinates at which density will be interpolated.
       Interpolated density value is returned in $result.
    ?
       Print current status
