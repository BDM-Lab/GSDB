structure {<structure-statement>} end


<structure-statement> :==

    data_cns_mtf {<mtf-record>}
       Read mtf records from file. Information from multiple files will
       be appended.
    reset
       Reset molecular topology database.
    ?
       Print information about current status.

       <mtf-record> contains molecular structure information in STAR format.
       <mtf-record>s are written by the write structure command. They are not
       stored in the rotating command buffer during a loop execution.
