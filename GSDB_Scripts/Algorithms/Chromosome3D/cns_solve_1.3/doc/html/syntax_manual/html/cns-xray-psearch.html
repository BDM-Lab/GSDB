psearch {<xray-psearch-statement>} end


<xray-psearch-statement> :==

    from=<real-space-object>
       no default
    level=<integer>
       Default=1
    nlist=<integer>
       List this number of highest peaks. Default=101
    fractional=<logical>
       Output list in fractional coordinates. Default=false
    symbol=<word>
       Symbols will be declared if a name is specified. No default.
    pstore=<integer>
       Store or restore peaks to/from peak store with given number. No default.
    reset
       Clear all peak stores.
