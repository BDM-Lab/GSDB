<xray-average-statement> :==

    from=<real-space-object>
       Real space object to be averaged. This will be overwritten by the
       averaged real space object.
    group {<xray-average-group-statement>} end
       Define a averaging group. The first group is the reference. Subsequent
       group definitions specify the transformation that superimposes
       group 1 on that group.
    manipulate {<xray-average-manipulate-statement>} end
       Rotate and translate real space objects.
    ?
       Print information about current status.
